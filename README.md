# deploy

Deploy repo for driving instructor website

1. Run npm run build
2. Upload the /dist folder to github repo by drag n drop. assets favicon.svg icons.svg index.html
3. Go to pages
4. build from github actions
5. select static html
6. Commit create static.yaml without modifying
7. Let the action finish.
8. If action is successful, your website is published.
9. In the custom domain add your domain name and save. It will check DNS if successful then force HTTPS.

That's All!
